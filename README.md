# End-to-End Sales Analytics Pipeline

## Overview
This project simulates a real-world retail analytics pipeline by extracting raw sales data from multiple sources, transforming it into clean business-ready tables, loading it into a SQL database, and visualizing key KPIs through an interactive dashboard.

## Business Problem
Retail businesses often struggle with fragmented data sources and lack visibility into:
- Revenue trends
- Customer behavior
- Product performance
- Shipping delays

This project solves that by building a centralized reporting pipeline.

## Tech Stack
- Python (Pandas, SQLAlchemy)
- PostgreSQL
- SQL
- Power BI / Tableau
- Git / GitHub
- Optional: Docker

## Architecture
Raw CSV Files → Python ETL → SQL Database → Dashboard
